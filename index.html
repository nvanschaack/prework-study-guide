HTML

  • HTML creates the content thats in the browser ex. text, images, layout
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Section elements are added to the body to organize sections by topic
  • Section elements are inside the main element

CSS

  • Used by developers to define how webpage content is displayed to users
  • Adds styling to content ex. aesthetics like font, color, visual layout
  • 3 ways to style a webpage with CSS: inline CSS, internal CSS style sheet, external CSS style sheet
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element

Git

  • Git is a software that tracks changes that you make to your code. It's version control for developers!
  • GitHub is a cloud-based service that allows developers to store, manage, and collaborate on code
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switched to it
  • git checkout main: gets you back to the main branch

JavaScript

  • Allows you to add logic/ability to interact with webpage
  • Control flow: order in which a computer executes code in a script. It uses "if" statements
  • A variable is a named container that allows us to store data in our code
  • An array is used to hold/store multiple related data items in a variable
  • An index is a list of numbers that indicates each items place on the list, always start with "0"
  • For loops consist of 1.Keyword "for", 2.Three-part conditional statement, 3.Code block
  • Functions are a set of instructions that tells a computer how to perform a certain task. Remember that to execute a function, we must "call it" in our code
